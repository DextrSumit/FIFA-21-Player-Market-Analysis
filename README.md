## 📌 Overview

This project provides a comprehensive exploratory data analysis (EDA) of the FIFA 21 player dataset. Using Python, Pandas, and Data Visualization tools, the analysis explores how player attributes—such as age, overall rating, potential, wages, preferred foot, nationality, and club—relate to market value and financial efficiency.

The project also includes a **Power BI Dashboard** and detailed business reports to provide actionable insights for player valuation, transfer market strategies, and club financial efficiency.

## 🎯 Objectives & Business Questions

This analysis is driven by key business questions aimed at uncovering market inefficiencies and identifying value:

1. **Market Value Drivers:** Which factors have the greatest impact on a player's market value?
2. **Age & Rating Correlation:** How do Age and Rating impact Market Value? When do players typically peak, and when does their value decline?
3. **Future Stars:** Which young players have the highest future potential?
4. **Undervalued Assets:** How can we identify undervalued gems, optimal investment values, and release clause inefficiencies?
5. **Wage Dynamics:** Are players with 5-star skill moves or weak foot paid more than equivalent players? Does versatility cost more?
6. **Financial Efficiency:** Which specific clubs are the most financially efficient (maintaining high ratings with low wage bills)?
7. **Player Traits:** Is there a financial premium for left-footed players? How do height and weight correlate with overall rating for specific positions?

_(Note: The full detailed exploration of 20 specific business questions is available in the Jupyter Notebook and the Analysis Report.)_

## 🚀 How to Run

1. **Clone the repository:**

   ```bash
   git clone <repository_url>
   ```

2. **Install Python dependencies:**

   ```bash
   pip install pandas numpy matplotlib seaborn jupyter
   ```

3. **Run the Analysis:**
   - Open `FIFA21_Player_Analysis.ipynb` using Jupyter Notebook.
   - Run all cells to execute the data cleaning, exploratory analysis, and visualizations.
4. **View the Dashboard:**
   - Open `FIFA21 Dashboard.pbix` using Microsoft Power BI Desktop to explore the interactive visual dashboard.

## 💡 Key Insights

- **Peak Age & Value:** Players generally reach their peak overall rating around age 27-29, after which market values begin to decline significantly.
- **Financial Efficiency:** Certain clubs manage to maintain high average player ratings while keeping their wage bills remarkably low, demonstrating superior transfer and scouting strategies.
- **Undervalued Assets:** The analysis successfully identifies young players with high potential and relatively low release clauses, highlighting optimal targets for club investments.
